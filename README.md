### Installation

##### Just, Clone this repository -
```sh
$ git clone https://github.com/Akshay-Arjun/69phisher.git
```

##### Change to cloned directory -
```sh
cd 69phisher
```
```sh
chmod 777 69phisher.sh
```
```sh
bash 69phisher.sh
```
- ### Dependencies

**`69phisher`** requires following programs to run properly - 
- `php`
- `wget`
- `curl`
- `git`
